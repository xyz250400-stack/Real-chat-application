# RealChat Pro - Real Time Chat App

Task 2 submission - Real-time chat application built with Flask + SocketIO

## Features
1. Nickname based login + Online users presence
2. Public rooms + Dynamic room creation  
3. Real-time messaging with timestamps
4. Sent status + Typing indicator
5. Auto-scroll to latest messages

## How to Run
1. Install: `pip install flask-socketio`
2. Run: `python app.py`
3. Open: `http://localhost:5000`

## Tech Stack
Flask, SocketIO, HTML, CSS, JavaScript
